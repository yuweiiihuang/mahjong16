## 首段

臺灣麻將是流行於臺灣的麻將打法。由於每人有十六張牌，亦稱「十六張麻將」。

## 遊戲器具

- 一百四十四張的麻將牌及三顆骰子  
- 常見配件：麻將桌、牌尺、記分紙、風向盤（圈風骰）

## 牌種

### 數牌

- 萬子：一萬至九萬  
- 餅子：一餅（一筒）至九餅（九筒）  
- 索子：一索（小鳥、雞）至九索（九條）

> 上述每種牌各四張，共 108 張數牌。

### 字牌

- 風牌：東、南、西、北  
- 三元牌：白、發、中

> 上述每種牌各四張，共 28 張字牌。

### 花牌

- 四季：春、夏、秋、冬  
- 四君子：梅、蘭、菊、竹

> 上述每種牌各一張，共 8 張花牌。  
> 部分地區不使用花牌（改為 136 張牌），南部較盛行。

## 遊戲進行

### 遊戲人數

四人。各坐正方桌四邊。

### 抽位

用以決定座位與東風位。

1. 先隨機就座。從牌中挑出東、南、西、北各一張，覆蓋並打亂。  
2. 隨機指定一位玩家擲三顆骰。依點數自擲者為 1 逆時針計數，決定抽牌順序。  
3. 同時決定東風位。南風坐東風下家，西風坐對家，北風坐上家。

**快速規則：點數 ÷ 4 取餘數**

| 餘數 | 對應 |
|---:|:---|
| 1 | 自己 |
| 2 | 下家 |
| 3 | 對家 |
| 0 | 上家 |

> 地方規則眾多，此處僅述常見流程。

### 洗牌

四人將 144 張牌覆蓋後推拉打散，稱「洗牌」（俗稱「乾泳」）。

### 堆牌

各自堆成高二底十八之二乘十八「牌牆」，四座圍成方形；若不玩花牌，底十七。此狀如城池，故稱「方城之戰」。

### 起莊、發牌

- 由東風位玩家擲骰，決定首局莊家與取牌起點。  
- 自莊家面前之 18 墩最右側起順時針數出骰點，留該墩，逆時針輪取。每次取兩墩（四張），四輪後各 16 張。  
- 例：若擲 11 點，則西風位為莊，莊家自第 12、13 墩取起；餘依序。

> 部分地區改為東位決定莊家，莊家再擲骰決定取牌起點。

### 非起莊發牌

- 起莊局：由「假莊」（東位）擲骰，決定莊家與取牌起點。  
- 非起莊局：由當局莊家擲骰，決定取牌起點。

### 開門、補花

- 發完牌後，莊家自取牌終點下一墩取一張，稱「開門」（門牌），此時莊家手牌 17 張。  
- 若有花牌，須一次全部攤出並自嶺上補足張數維持 17 張；閒家依序補至 16 張。  
- 補到花牌需「後補」（續補）至無新花為止。

### 捨牌、摸牌或叫牌

- 莊家未胡則捨出一張至牌河。  
- 閒家可選：吃、碰、槓、摸、胡。槓後自牌尾補一張再捨一張。胡則結束本局。

### 流局

- 牌牆剩「八墩」（16 張）未有人胡，宣告流局。  
- 地方差異：有的一槓留一張或留一墩；也有取至不剩才結束。  
- 流局時莊家不變（臭莊、臭連）。

### 特殊流局

常見兩例：
- **四家開槓**：合計槓數超過四個流局。  
- **四風連打**：四家依序打出相同風牌流局。  

多數玩法不採用特殊流局。

### 下莊

若非莊胡且非流局，則下家上莊。

### 時間單位

- **局**：出現一次胡牌或流局。  
- **圈**：四人輪流坐莊為一圈。第一圈東風圈，第二圈南風圈，以下類推。  
- **將／雀**：四圈為一將（一雀）。一般一雀約 90 分鐘；兩將約 3 小時。

### 再抽位

一雀結束續玩需重抽。通常規定若抽後相對位置不變，需重抽。

## 胡牌

### 順子

連續三張數牌如「一二三萬」「五六七索」「三四五筒」。合計 21 種。

### 刻子

三張相同的牌如「一一一萬」「東東東」「八八八筒」。  
- **明刻**：經碰牌組成，需攤出。  
- **暗刻**：未經碰牌組成，無需攤出。  
- 槓子（明槓、暗槓）屬刻子的一種。  
合計 34 種。

### 面子

刻子與順子合稱面子。

### 雀頭

又稱眼、將、對子。兩張相同的牌，如「東東」「一一萬」。合計 34 種。  
（此處「將」與香港麻將用法不同。）

### 胡牌方式

- **榮和（放槍）**：胡他人捨出的牌。多人同時可胡時，離放槍者最近者優先；一般無一炮雙響。  
- **自摸**：自摸到成和之牌。  
- **標準和型**：五個面子加一個雀頭。

### 過水與過手

- **過水**：可胡而不胡，需等過手後方可再胡。  
- **過手**：需摸入一張**不能胡**之牌並捨出一張後，方可胡。若摸入即為自摸也不可，需再過一手；否則屬炸胡。  
臺灣麻將對過水規定較嚴格。

### 詐胡

未達五面一眼即宣告胡，屬違規。一般需受點數處罰。

## 胡牌計分

### 底、台

- 例：**30/10** 表示一底 30，一台 10。**200/50** 表示一底 200，一台 50。  
- 結算時以「底 + 台」換算為點數。

### 胡牌與自摸

- 榮和：向放槍者收取「一底 + 台」。  
- 自摸：向三家各收取「一底 + 台」。  
- 同形牌，自摸得分約為榮和的兩倍多。

### 台數表

> 各地牌風差異大，下表為常見定義；有的地區設「滿台 8 台」上限。

| 台種 | 台數 | 敘述 |
|---|:---:|---|
| **莊家** | 1 | 做莊者，無論胡或放槍皆多算一台。莊胡或流局可連莊。 |
| **連莊拉莊** | 2N+1 | 連一拉一為三台，連二拉二為五台，以下類推。 |
| **門清** | 1 | 無吃、無碰、無槓。全部牌皆自摸所得；僅胡牌那張可為他人捨出。 |
| **獨聽（邊張／中洞／單吊）** | 1 | 僅一種牌能胡。例：一二萬聽三萬；八九筒聽七筒；二四索聽三索；五七萬聽六萬；或單吊一筒、發。因他家打出而**才**成為單一待張者，不算。 |
| **圈風牌** | 1 | 當圈風之刻（或槓）。 |
| **門風牌** | 1 | 自家門風之刻（或槓）。 |
| **正花** | 1 | 依開門位置逆時針之花位：東（春、梅）、南（夏、蘭）、西（秋、菊）、北（冬、竹）。 |
| **三元牌** | 1 | 胡牌時，中／發／白任一刻（或槓）。 |
| **自摸** | 1 | 自摸成和。 |
| **河底撈魚** | 1 | 流局前最後一名玩家的最後一張捨牌被胡。 |
| **海底撈月** | 1 | 牌牆最後一張自摸胡。可同時加自摸台。 |
| **槓上開花** | 1 | 因摸到花而加槓或暗槓後補牌即胡。可同時加自摸台。 |
| **搶槓** | 1 | 加槓視同捨出，供他家榮胡。 |
| **不求** | 3 | 門清 + 自摸。 |
| **全求** | 2 | 胡時手牌只剩一張，且單一待張；其餘皆已吃／碰／槓。 |
| **花槓** | 2 | 集齊「四季」或「四君子」。可加算正花。 |
| **平胡** | 2 | 無花、無字、無刻、非獨聽、非自摸之榮胡。 |
| **三暗刻** | 2 | 胡時有三個暗刻（非碰所得）。 |
| **混一色** | 4 | 單一門數牌 + 字牌。 |
| **碰碰胡** | 4 | 全刻無順，且未吃。 |
| **小三元** | 4 | 中發白兩刻一眼。不得重複計算三元台。 |
| **四暗刻** | 5 | 胡時四個暗刻。不得重複計算三暗刻。 |
| **清一色** | 8 | 全部為單一門數牌。 |
| **四槓牌** | 8 | 單家自成四槓即視為自摸胡，本局結束（不加自摸台）。若由兩家以上合計四槓則為和局。 |
| **五暗刻** | 8 | 手上五個暗刻。不得與其他暗刻台重複。 |
| **大三元** | 8 | 中發白三刻。不得重複計算三元台。 |
| **小四喜** | 8 | 東南西北三刻一眼。可另加圈風、門風之風牌台。 |
| **地聽（咪機）** | 8 | 無任何吃、碰、（暗）槓情況下，每家過兩手（八張）內宣告聽牌，且當局內須胡牌；宣告後不得換牌或過水。 |
| **七搶一** | 8 | 集任七張花，視為胡另一位持花者。 |
| **八朵花** | 8 | 集八張花，視為自摸 8 台。 |
| **大四喜** | 16 | 東南西北四刻。風牌台（圈風、門風）不再重複。 |
| **天聽** | 16 | 莊取完牌、未開門即聽，需告知。 |
| **地胡** | 16 | 開局後，莊家捨出的第一張牌即被胡。 |
| **天胡** | 32 | 開局後，莊家未捨牌即胡。 |
| **詐胡** | ？ | 未成和卻宣告胡；或過水未過手即胡。需一家賠三家，賠台數依詐胡所宣稱之台計（例：宣告地聽後詐胡，賠一底八台）。 |
| **各地** | ？ | 各地台數不同，應先約定。有地區設滿台 8 台上限。 |

## 各項說明

### 獨聽說明

- **單騎**（吊將）：缺眼單張聽眼。  
- **嵌張**（中洞、卡張）：如一三聽二、二四聽三、六八聽七、七九聽八等。  
- **邊張**（站壁）：一二聽三或八九聽七。  
- 亦有玩法採「抓得出來就有台」之處理（例以拆解方式視為單騎）。

### 正花、正風說明

- 擲骰同時決定風向與正花位：  
  - 東位：東風、壹花（春、梅）  
  - 南位：南風、貳花（夏、蘭）  
  - 西位：西風、參花（秋、菊）  
  - 北位：北風、肆花（冬、竹）  
- 例：東風圈且自家門風為東，拿東刻稱「東風東」，風台 +1。雙壹花亦然。

### 全求說明

- 五個面皆為明組合，暗手僅剩一張且單騎。  
- 變體一：僅**榮胡**才算全求；若自摸稱「半求」並另加 1 台。  
- 變體二：全求 2 台，另加單騎 1 台共 3 台。

### 平胡說明

口訣：「無花、無字、無刻、先有眼、聽雙頭、非自摸」。各地常見限制：
1. 門清無平胡。  
2. 若有其他台種則無平胡（如搶槓胡、清一色）。  
3. 聽超過兩種牌無平胡（由「聽雙頭」字面解釋）。  
4. 手牌含三張相同牌無平胡（由「無刻」解釋）。  
可參考日本麻將之「平和」無符要求，或香港麻將之「平胡」對全數牌、無刻之要求。

### 字一色說明

字一色出現率低於大四喜。台數設定常見兩派：  
- 設為 **16 台**，且可複合碰碰胡、大小三元四喜、正字等，合計可逾 24 台。  
- 視為清一色 **8 台**，但可與其他字台複合，合計仍可超過 16 台。  
也有規定字一色 16 台但**不可**複合碰碰胡與其他字台之變體。

### MIGI、天、地、人胡

此類獎勵「天運」與快速宣告／成和。  
- **MIGI**：無吃碰槓之門清狀態，前兩次（或一次）摸牌以前宣告聽牌。宣告時機常見兩式：  
  1. 摸牌**前**任何時候（易與他人叫牌衝突）。  
  2. 摸牌**後**捨出時（若前已發生叫牌可能無法宣告）。  
- 有玩法將起手聽牌與摸後聽牌區分為天聽／地聽；或再分莊家／閒家。  
- 天胡、地胡台數亦有高低之爭，考慮公平性，多處採相同台數。

### 花胡之說明

- **七搶一**與**八朵花**兩類。七搶一是否可不明示第八花以規避放槍具爭議，部分牌風不採此類台種或乾脆不玩花胡。

### 特殊胡牌

- **搶槓**：甲聽牌中，乙先前碰得該張又摸同張宣告加槓時，雖未捨出亦可被甲榮胡，並加計搶槓台。  
- **四槓胡**：同一家自成四槓視為自摸胡，該局結束；如由不同玩家合計四槓則為和局並計連莊。
